### 功能

从超声机上采集的图片往往存在一些型号、深度等信息，本项目能够自动从超声图中裁剪超声成像区域。

### 使用说明

1. 拷贝US_SSD至本地
2. 读取权重
3. 设置类别为2（背景+超声区）
4. 调用extract_us_region
5. 样例代码如detect.py的main函数中，假设路径为---./
  ​							---data
  ​							---US_SSD
  ​							---yourcode.py

### tips

1.一般情况下，置信度>0.99，在我本地数据集中，13000例超声图，<0.99的有160个
